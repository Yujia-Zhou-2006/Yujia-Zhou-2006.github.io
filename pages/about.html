About this website

This site is built with Jekyll, Primer CSS, and custom responsive design. It is fully open source and designed for both desktop and mobile, with a focus on clean layout, accessibility, and easy content maintenance. All projects and blog posts are managed via Markdown files and site configuration, so you can update content with minimal effort.

Front-end filtering, theming, and other features are implemented with lightweight JavaScript and CSS. If you have any questions or suggestions, feel free to reach out via the social links in the footer.
